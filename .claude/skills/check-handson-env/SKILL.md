# 目的

ハンズオンカフェ (2026-06-24) に参加するための事前条件を、ターミナルで確認してください。
以下の4項目すべてを順番にチェックし、結果をまとめて報告してください。

# チェック項目と確認手順

## 1. Antigravity / AI コーディングツールが動いている

このメッセージを読めている時点で ✅ です。そのまま次に進んでください。

## 2. GitHub に push できる (PAT またはブラウザ認証)

1. `gh auth status` を実行する
2. ログイン済みかどうか、git プロトコル (HTTPS/SSH) の設定を確認する
3. `gh repo list --limit 3` を実行して、自分名義のリポジトリが1つ以上あるか確認する

**NG の場合の案内:**
- GitHub CLI 未インストール → `winget install GitHub.cli` (Windows) / `brew install gh` (Mac)
- 未ログイン → `gh auth login` を対話的に実行
- リポジトリがない → `gh repo create my-first-repo --public --clone` で作成

## 3. ブラウザ版 Claude にログインできる

ターミナルからは直接確認できないため、以下のメッセージをユーザーに表示する:

> ブラウザで https://claude.ai を開き、ログインできるか確認してください。
> ログインできたら OK です。

## 4. Zenodo Sandbox にログインできる

ターミナルからは直接確認できないため、以下のメッセージをユーザーに表示する:

> ブラウザで https://sandbox.zenodo.org/ を開き、GitHub アカウントでログインできるか確認してください。
> ログインできたら OK です。

# 結果の報告

チェック結果を以下の形式でまとめてください:

```
🔍 環境チェック結果
─────────────────────────
1. AI ツール:        ✅ / ❌
2. GitHub push:      ✅ / ❌  (ユーザー名: xxx)
3. ブラウザ版 Claude: 👆 手動で確認してください
4. Zenodo Sandbox:   👆 手動で確認してください
─────────────────────────
```

- すべて ✅ (手動確認を除く) なら「ハンズオンの準備完了です 🎉」と伝える
- ❌ がある場合は、具体的な解決コマンドを案内する
